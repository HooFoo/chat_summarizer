# Telegram Bot Summary Manager

Этот бот предназначен для работы в многопользовательских чатах. Он сохраняет все сообщения в базу данных и позволяет генерировать отчеты (саммари) с помощью Gemini AI.

## Возможности
- Логирование всех сообщений (текст, медиа, ответы).
- Генерация общего саммари за указанный период (по умолчанию 30 дней).
- Специализированные отчеты по пользователям и задачам.
- Поддержка Webhook.

## Установка
1. Создайте базу данных MySQL и выполните скрипт `db.sql`.
2. Скопируйте файлы на ваш сервер с поддержкой PHP.
3. Отредактируйте `config.php`, указав:
   - `TELEGRAM_BOT_TOKEN` (получите у @BotFather)
   - Параметры подключения к БД (`DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASS`)
   - `GEMINI_API_KEY` (получите в Google AI Studio)
4. Установите Webhook для вашего бота:
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/setWebhook?url=https://your-domain.com/webhook.php`

## Команды
- `/summary [дата]` - Общее саммари (например, `/summary 22.01.2025`). Если дата не указана, берется 30 последних дней.
- `/summary_users [дата]` - Отчет по активности пользователей.
- `/summary_tasks [дата]` - Отчет по задачам и поручениям.

## Требования
- PHP 7.4+
- MySQL
- Расширение cURL для PHP
- Расширение PDO MySQL для PHP
